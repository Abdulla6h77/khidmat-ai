# KhidmatAI Artifacts

This folder contains all Antigravity agent traces, workplans, decision logs, and screenshots required for the AI Seekho Hackathon Challenge 2 submission.

## Folders
- `agent_logs/`: Raw logs from the agent pipeline.
- `screenshots/`: Screenshots of the application and agent steps.
- `traces/`: Execution traces and metric logs.
- `antigravity_workplans/`: Generated workplans and decision documentation.
